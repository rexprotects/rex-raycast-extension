# Rex - Raycast Extension

Control Rex for MacBooks directly from Raycast.

## Features

- **Arm Rex**: Quickly arm Rex
- **Disarm Rex**: Quickly disarm Rex
- **Get Rex Status**: Check the status of Rex

## Requirements

- Rex installed on your Mac

## Installation

1. Install dependencies:

   ```bash
   npm install
   ```

2. Build and run in development:
   ```bash
   npm dev
   ```

## Usage

Once installed, you can access the commands from Raycast:

- Type "Arm Rex" to arm your security system
- Type "Disarm Rex" to disarm your security system
- Type "Get Rex Status" to check the current status
